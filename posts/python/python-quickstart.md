# Python 快速入门

Python 以其简洁的语法和强大的生态闻名。

## 为什么选择 Python？

- **语法简洁**：用最少的代码表达想法
- **生态丰富**：数据分析、机器学习、Web开发应有尽有
- **门槛低**：适合初学者入门编程

## 第一个程序

```python
def greet(name: str) -> str:
    return f"Hello, {name}!"

if __name__ == "__main__":
    print(greet("World"))
```
