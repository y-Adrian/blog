# 开发工具推荐

好的工具能让开发效率翻倍！

## 代码编辑器

- **VS Code** - 轻量、插件丰富
- **Neovim** - 高效、可定制
- **JetBrains 全家桶** - 专业、功能强大

## 终端工具

- **iTerm2** - macOS 终端替代品
- **Oh My Zsh** - 美化终端
- **tmux** - 终端多路复用

## Git 图形化工具

- **GitHub Desktop** - 简单易用
- **Sourcetree** - 功能丰富
- **lazygit** - 终端内的 Git 界面
