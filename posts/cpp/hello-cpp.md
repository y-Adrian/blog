# C/C++ 入门指南

欢迎来到 C/C++ 编程的世界！

## 为什么学习 C/C++？

- **性能强大**：直接操作内存，适合游戏、嵌入式、系统编程
- **底层控制**：理解计算机工作原理的最佳选择
- **广泛应用**：操作系统、编译器、游戏引擎、数据库

## 第一个程序

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

> 学习编程的最好方式就是动手实践。
